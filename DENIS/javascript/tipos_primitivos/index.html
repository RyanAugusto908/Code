Tipos primitivos

Vamos entender o que são tipos primitivos

Eles são :

  • String

    • Tipo String
    O tipo String do JavaScript é usado para representar dados textuais. Cada elemento na String ocupa uma posição na String. O primeiro elemento está no índice 0, o próximo no índice 1 e assim por diante. O comprimento de uma String é o número de elementos nela.

    Ex: Azul, Carlos, Idade...

  • Number

    • O tipo Number é capaz de armazenar números de ponto flutuante positivos e negativos.

    Ex: 7, 1.5, -8, 1.5, 0.2, 7.9

  • Boolean

    • Boolean representa uma entidade lógica e pode ter dois valores: true e false.

    Ex: true false

  • Null

    • O tipo null tem exatamente um valor: null
    valor nulo ou vazio.

  • Undefined

    • Uma variável que não recebeu um tipo definido, undefined ou seja indefinido.

Para identificar o tipo da variável utilizamos o comando (typeof) + variável
